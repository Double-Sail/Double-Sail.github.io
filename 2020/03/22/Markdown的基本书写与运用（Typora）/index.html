二、不带快捷书写操作与测试

三个` :==代码块的输入==。

代码块输入测试

#hello python
print('hello world!')
//java 代码块输入
package demo04;

import java.util.Arrays;

//请使用 Arrays 相关的API，将一个随机字符串中的所有字符升序排列，并倒序打印。
public class Demo02ArraysTest {
    public static void main(String[] args) {
        String str ="sadfasfwqfwqdfs";

        //如何进行升序排列：sort
//        必须是一个数组，才能使用Arrays。sort方法
//        String-->数组，用toCharArray
        char[] chars = str.toCharArray();
        Arrays.sort(chars);

        //需要倒叙排列.chars.forr
        for (int i = chars.length - 1; i >= 0; i--) {
            System.out.print(chars[i]);
        }
    }
}

==无序、有序、任务列表==：

无序：+ 空格；

  • 111（直接一次回车，可以自动生成下一个同级无需标点）

    • 222(按下Tab，可以转化为二级，同理可以转化后续等级）

    （二级生成后，按两次回车既可以取消本级的无序标点，变成普通文本输入格式）

  • 111（三次回车可以回到上一级无序标点）

有序：输入相应数字+空格；

  1. 111
  2. 222

任务列表：- [ ]；（方框可以进行勾选操作，勾选后，源码会有一定的改变）；

  • 111
  • 222

==参考链接==：[^具体数字]；

应对恐惧最好的办法就是勇敢的面对他，加油，奥利给！^1（再点击这个标志，即可以生成相应格式，再输入来源）

==行内链接==：链接文字都是用 [方括号] 来标记，使用 (圆括号) 来把文字转成链接。还可以为链接文字配个Title，当然 Title 属性是可选项，加不加看心情咯。

[链接文字](链接网址 "标题")
This is an [example link](http://example.com/ "With a Title").

This is an example link.

==上标和下标==：上标^具体数字^,下标~具体数字~； x^2^ H2O

==高亮==：==文本输入==；

==内联公式==：$公式$； $e^2$

==分割线==：---；

==emoji==：英文冒号+要求表情+后面可以输入解释性文字； :arrow_up:上面是分割线内容； :smile:这是笑脸
